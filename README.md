  <b>Установка</b>: <br>Открыть AIMP, затем  "<i>Настройки</i>", затем "<i>Ассоциация с файлами</i>" и нажать "<i>Плюсик</i>" внизу, <br>следом выбрать архив с иконками, нажать "<i>Открыть</i>" и "<i>Применить</i>", всё!
